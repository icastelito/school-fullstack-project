## Colégio Astra
Este é um projeto full-stack desenvolvido utilizando Django e React. O sistema é um cadastro de escola denominado Colégio Astra.

### Como rodar o projeto

#### Para rodar o backend, execute o seguinte comando dentro da pasta backend:
docker-compose build

#### Para rodar o frontend, execute o seguinte comando dentro da pasta frontend:
npm install
npm run dev

### Tecnologias utilizadas
Django
React
Docker
Funcionalidades
Cadastro de escolas
Cadastro de alunos
Cadastro de professores
Listagem de escolas
Listagem de alunos
Listagem de professores
